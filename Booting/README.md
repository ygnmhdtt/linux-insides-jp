* [video mode 初期化とプロテクトモードへの遷移](linux-bootstrap-3.md) - カーネルセットアップコード内のビデオモード初期化と、プロテクトモードへの遷移について

* [カーネルの展開](linux-bootstrap-5.md) - カーネルの展開の前の準備と直接的な展開の詳細について
